Installation manuelle de secours

FFmpeg est un projet libre indépendant. En cas d’échec de l’installation automatique, privilégiez la page officielle, qui référence les fournisseurs de binaires adaptés à chaque système.

  1. Téléchargez FFmpeg uniquement depuis une source référencée par le projet officiel.
  2. macOS : Homebrew reste la méthode recommandée pour les Mac Apple Silicon.
  3. Windows : le paquet Essentials de gyan.dev suffit pour Centauri Live.
  4. Relancez l’application, puis exécutez de nouveau le diagnostic.
